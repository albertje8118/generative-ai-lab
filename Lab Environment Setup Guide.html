Lab Setup Guide

This guide will walk you through setting up your environment to run the lab for the Generative AI course. We will be using Visual Studio Code, Python with Anaconda, and Hugging Face's transformer models.

Hardware and Software Requirements

Hardware

  • Laptop with integrated graphics
  • Minimum 8GB RAM
  • Minimum 20GB free disk space

Software

  • Operating System: Windows, macOS, or Linux
  • Visual Studio Code
  • Anaconda Distribution with Python 3.x
  • Hugging Face Transformers library

Step-by-Step Lab Environment Setup

Step 1: Install Visual Studio Code

Download and install Visual Studio Code from the official website: https://code.visualstudio.com/

Step 2: Install Anaconda

Download and install Anaconda Distribution from the official website: https://www.anaconda.com/

Step 3: Set Up a Python Environment

conda create -n generative-ai-lab python=3.9
conda activate generative-ai-lab

This command creates and activates a new environment named generative-ai-lab with Python 3.9.

Step 4: Install Required Python Packages

conda install -c conda-forge transformers
conda install -c conda-forge python-dotenv

This installs the Hugging Face Transformers library and the python-dotenv package.

Step 5: Set Up Your Project in Visual Studio Code

  • Open Visual Studio Code.
  • Create a new folder for your project and open it in Visual Studio Code.
  • In the terminal, activate your Conda environment: conda activate generative-ai-lab.

Step 6: Create a .env File to Store Your API Key

Create a file named .env in your project directory to securely store your Hugging Face API key.

HUGGINGFACE_API_KEY=your_huggingface_api_key_here

Replace your_huggingface_api_key_here with your actual API key from Hugging Face.

Step 7: Write Your Python Script

Create a Python script (e.g., summarize_dialogue.py) and use the following template:

import os
from dotenv import load_dotenv
from transformers import pipeline

# Load environment variables from the .env file
load_dotenv()

# Access the Hugging Face API key
api_key = os.getenv('HUGGINGFACE_API_KEY')

# Example usage: creating a summarization pipeline
summarizer = pipeline("summarization", model="facebook/bart-large-cnn")

dialogue = """
Customer: I have a problem with my order.
Support: I'm sorry to hear that. Can you provide more details?
Customer: I received the wrong item.
Support: We apologize for the inconvenience. We'll send the correct item right away.
"""

summary = summarizer(dialogue, max_length=50, min_length=25, do_sample=False)
print(summary[0]['summary_text'])

This script loads the API key from the .env file and uses it to create a summarization pipeline using the Hugging Face Transformers library.

Step 8: Run Your Script

In the Visual Studio Code terminal, ensure your Conda environment is activated, then run your script:

python summarize_dialogue.py

The script will output the summarized dialogue.
